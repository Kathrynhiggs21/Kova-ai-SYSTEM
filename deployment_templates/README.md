# Deployment Templates for Kova AI Repositories

This directory contains templates for deploying and configuring all Kova AI repositories.

## Structure

```
deployment_templates/
├── common/              # Shared configs for all repos
├── kova-ai/            # Backend service
├── kova-ai-site/       # Website/docs
├── kova-ai-mem0/       # Memory system
├── kova-ai-docengine/  # Document engine
└── Kova-AI-Scribbles/  # Experimental
```

## Usage

1. Copy appropriate template to your repo
2. Customize configuration values
3. Run deployment script
4. Verify services are running

## Templates Included

- Docker configurations
- CI/CD pipelines
- Environment setup
- Service configs
- Deployment scripts
